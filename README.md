# Ailaysa Tiger Translation Dashboard

A modern, responsive translation management dashboard built with Next.js, Tailwind CSS, and TypeScript.

## How to run locally

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to `http://localhost:3000` to view the application.
